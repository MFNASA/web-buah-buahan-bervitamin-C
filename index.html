Hallo Sobat sehat, apakah kalian tau vitamin C pada buah-buahan itu sangat penting loh,Yuk simak baik-baik

Vitamin C adalah nutrisi penting yang ditemukan dalam berbagai macam buah-buahan. Vitamin ini sangat penting untuk sistem kekebalan tubuh, kesehatan kulit, dan banyak fungsi tubuh lainnya. Di website ini, Anda akan menemukan informasi tentang buah-buahan yang kaya akan vitamin C dan manfaatnya untuk kesehatan.

Fakta Menarik

Vitamin C membantu penyerapan zat besi dalam tubuh dan berperan penting dalam perbaikan jaringan.

Konsumsi Harian

Dianjurkan untuk mengonsumsi setidaknya 65-90 mg vitamin C setiap hari untuk orang dewasa.

Buah Musiman

Banyak buah kaya vitamin C tersedia sepanjang tahun, seperti jeruk, kiwi, dan pepaya.

Buah-buahan dengan Vitamin C Tinggi

Jeruk

Jeruk kaya akan vitamin C dan sangat baik untuk kesehatan kulit dan sistem kekebalan tubuh.

Kiwi

Kiwi mengandung lebih banyak vitamin C daripada jeruk, membantu meningkatkan kekebalan tubuh.

Stroberi

Stroberi kaya antioksidan dan vitamin C, baik untuk kesehatan jantung dan kulit.

Pepaya

Pepaya kaya vitamin C dan serat, baik untuk pencernaan dan kesehatan kulit.

Nanas

Nanas mengandung vitamin C serta bromelain, enzim yang baik untuk pencernaan.

Mangga

Mangga mengandung vitamin C dan A, baik untuk kesehatan mata dan kulit.

Jambu Biji

Jambu biji memiliki kandungan vitamin C yang sangat tinggi, baik untuk kekebalan tubuh.
